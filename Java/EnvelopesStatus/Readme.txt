Readme.txt for Docusign SDK Java EnvelopeStatus sample.
===========================================================================

System Requirements:
-	Java Development Kit 1.6
-	Apache Ant 1.6
-	Tomcat 6 (or equivalent)

This sample code requires a DocuSign DevCenter account. If you do not
already have a DevCenter account please go to
http://www.docusign.com/devcenter/ and sign up for one. This sample will
not function without a valid DevCenter account.

Overview:
---------------------------------------------------------------------------

EnvelopeStatus Web Application is optimized for iPhone. It’s a sample that
retrieves your API credentials using DocuSign’s new Login web service. The
interface presents a status of all the sent out envelopes and allows the
sender to void envelopes en route, retrieve current PDF and send out a
reminder. 

How to Build: 

1. Run Apache ANT in the project root.  

You can deploy the WAR file to your web server.
