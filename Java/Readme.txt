Readme.txt for Docusign SDK Java samples

System Requirements:
-	Java Development Kit 1.6
-	Apache Ant 1.6
-	Tomcat 6 (or equivalent)

Overview:

Each sample includes the source code for the services required to execute the
applications. All of the appications run as tomcat servlets bundled as war 
files. Refer to the Readme.txt in the root directory of each sample for further
details.
