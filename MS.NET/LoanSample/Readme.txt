Readme.txt for Docusign SDK MS.NET C# LoanCo sample.

System Requirements:
-	Visual Studio 2008+ (with applicable system requirements)
-	NUnit 2.5.7+ (from http://www.nunit.org/?p=download)

Overview:

SampleLoanCo solution contains two projects. One project which builds Signing.dll 
is a small wrapper around DocuSign Web Service Proxy. The second project is a small 
LoanCo web application. You can see this application working on the DocuSign sample 
server http://samples.docusign.com/loanco/.

The solution file for loading, building, and executing this project is at 
./SampleLoan.sln.

More details can be found at the PDF in the docs directory.
