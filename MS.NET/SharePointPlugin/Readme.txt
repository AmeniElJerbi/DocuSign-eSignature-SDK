Readme.txt for Docusign SDK MS.NET C# Sharepoint plugin sample.

System Requirements:
-	Visual Studio 2008+ (with applicable system requirements)
-	NUnit 2.5.7+ (from http://www.nunit.org/?p=download)
-	.NET 2.0+, Windows SharePoint Services 3.0

Overview:

The SharePoint Solution integrates DocuSign services with SharePoint 2007.

The solution file for loading, building, and executing this project is at 
./DocuSignSignature_SourceCode.sln.

More details can be found at ./DocuSignSignature_SourceCode/SETUP/readme.html.
