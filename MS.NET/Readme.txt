Readme.txt for Docusign SDK MS.NET samples

System requirements for Microsoft .NET samples:
-	Visual Studio 2008+ (with applicable system requirements)
-	NUnit 2.5.7+ (from http://www.nunit.org/?p=download)

Overview:

Each sample includes the source code for the services required to execute the
applications. All of the appications include a Microsoft Visual Studio solution file
that can perform building, deployment, and debugging.
