DocuSign Web Services API 3.5 SDK PHP Samples

System requirements for PHP samples:
-	PHP Version 5.3+
- 	mcrypt/2.5.7 libcurl/7.19.4 OpenSSL/0.9.8k zlib/1.2.3 

OverView:

Each sample includes the source code for the services required to execute the 
application. Deployment is just copying the files into a directory on your server.
Refer to the Readme.txt in the root directory of each sample for further details.
